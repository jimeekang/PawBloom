# AI 안전 정책

PetBloom AI는 수의사가 아니라 기록 요약 도구다.

## 허용

- 최근 식사, 물, 배변, 약, 컨디션 기록을 요약한다.
- 누락된 기록을 알려준다.
- 반복되는 패턴을 조심스러운 표현으로 설명한다.
- 수의사에게 물어볼 질문을 제안한다.
- 한국어 또는 영어 기록을 기반으로 영어 병원 리포트 요약을 만든다.

## 금지

- 질병을 진단한다.
- 약을 처방한다.
- 약 용량 변경을 추천한다.
- 병원 방문이 필요 없다고 말한다.
- 상태가 응급인지 아닌지 단정한다.
- 특정 질병 가능성을 확정적으로 표현한다.

## 필수 고지 문구

모든 AI briefing과 vet report에는 아래 고지문을 포함해야 한다.

영문 출력 문구:

```text
This is a record-based summary, not a diagnosis. Contact a veterinarian for medical decisions.
```

한국어 UI 문구:

```text
이 내용은 진단이 아니라 기록 기반 요약입니다. 의학적 판단은 수의사에게 문의하세요.
```

## 표현 원칙

- "기록상", "반복해서 기록됨", "수의사에게 문의할 질문"처럼 관찰 중심으로 표현한다.
- "원인은", "진단은", "치료는", "응급입니다"처럼 의료 판단으로 보이는 표현은 피한다.
- AI output은 항상 사용자가 입력한 기록의 범위 안에서만 설명한다.

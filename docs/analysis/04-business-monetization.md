## A. CEO 관점 — 돈이 되는가

### 수익성 판정: 조건부 Pivot. 현 형태 그대로면 Kill에 가깝다.

| 시나리오 | 12~18개월 흑자 확률 | 조건 |
|---|---|---|
| 현 상태 유지(순수 기록 구독) | **~15%** | 두 시장 모두 순수 기록 구독은 전환 ~2%. 한국은 촉촉한코가 무료로 같은 걸 함 |
| 이중언어 wedge 실구현 + 만료링크 배선 | **~30%** | 재한 외국인/재호 한인 저경쟁 니치 점유. 단 TAM 천장 낮음 |
| 위 + 진짜 LLM 브리핑 + 커머스/보험 리드 레이어 | **~40%** | AI 리포트 선점 창이 닫히기 전(6~12개월) 속도전 성공 시 |

(흑자 확률 15/30/40%는 근거 모델 없는 점추정 — [00](00-executive-summary.md) 미검증 수치 원장 참조.)

핵심 근거: **구독만으로는 두 시장 다 안 된다.** 한국은 순수 구독 저항 강함 + 커머스 결합 우세(흑자 사례 페오펫·핏펫은 전부 거래 레이어 보유), 호주도 프리미엄 전환 median 2.1%. 돈은 "기록"이 아니라 "기록이 유발하는 거래(약·검사·보험·예약)"에 붙는다.

### 티어 재설계 (현재 구조는 결제를 유도하지 못함)

현재(`entitlement.ts`): Free = 펫1·AI✕·공유✕ / Plus = 펫5·AI○ / Family = 펫10·AI○·공유○.

세 가지 문제: ① 최강 유료 후크(AI 브리핑)가 실물 없음(결정론적 템플릿), ② 가족공유는 이미 무료 표준(Petmaid·PetTimely), ③ maxPets(1→5→10)는 대다수가 1~2마리라 유인 약함.

**재설계 (결제 유발점 중심):**

| | Free | Plus | Family |
|---|---|---|---|
| 케어 다이어리 | 무제한 | 무제한 | 무제한 |
| 만료링크 vet 공유 | 월 1회 | 무제한 + 만료기간 설정 | 무제한 |
| **LLM 병원 브리핑** | ✕ | **○(핵심 후크)** | ○ |
| **영/한 이중언어 리포트** | ✕ | ✕ | **○(Family 전용 후크)** |
| 가족 멤버 | 1인 | 1인 | 6인 |
| 리포트 보관 이력 | 최근 7일 | 무제한 | 무제한 |

**가격 (벤치마크 근거):**
- 호주 Plus: **A$6.99/월 또는 A$49/년** (7일 무료체험 필수 — 트라이얼이 프리미엄 대비 ~5배 전환).
- 호주 Family: **A$89/년** (6인 = 1인당 A$15로 방어 쉬움).
- 한국 Plus: **₩4,900/월 또는 ₩39,000/년** — 단 구독 저항 커서 핏펫 멤버십 ₩2,900/월이 벤치마크. **한국은 구독 최소화(₩3,900 이하)하고 커머스 로열티로** 가는 게 현실적.

### 단위 경제

진짜 LLM 도입 가정: 브리핑 1건 ≈ US$0.0036(≈₩5, 무시 가능), Plus 월 8회 ≈ US$0.03/유저. 사진 스토리지 연 US$0.05 + 전송 US$0.1~0.3. **유저당 연 원가 ≈ US$0.5~1.5, Plus 연 A$49 기준 마진 95%+.** 소프트웨어 단위경제는 훌륭하다 — **문제는 원가가 아니라 매출(전환)이다.** AI 원가가 싸다는 건 Free에도 "월 1회 진짜 브리핑"을 미끼로 뿌려 퍼널을 넓힐 여유가 있다는 뜻. (단가·빈도·마진은 가정값 — [00](00-executive-summary.md) 참조.)

> 주의: "clinician-grade" 정확도로 Opus급+RAG를 쓰면 브리핑당 비용 10~30배(US$0.05~0.1). 그래도 유저당 연 US$5라 A$49 구독이면 견딤. 모델 등급은 마진이 아니라 규제 리스크로 결정.

### TAM / SAM / SOM (러프)

- **호주 TAM:** 반려가구 790만 × 지불의사 15%(임의 계수) ≈ 120만.
- **초기 웨지 SAM(재호 한인/아시아계):** 현실적 도달가능 ≈ 3만~8만 가구.
- **SOM 1년차:** 웨지의 2~5% → 다운로드 8천~1.5만, MAU 2천~4천, 유료전환(트라이얼 경유) 15% → 유료 300~600명 → **연매출 A$18K~45K.** 즉 **1년차는 사업이 아니라 검증 단계.**
- 두 시장 웨지 합쳐도 1년차 유료 ~1,000명 미만. 린 팀(1~2인) 사이드로는 유지 가능하나, 큰 사업이 되려면 웨지를 넘어 영어권 대중시장으로 확장해야 성립.
- (지불의사 15% 임의 계수가 모든 SAM/SOM/매출 곱셈 기반 — [00](00-executive-summary.md) 미검증 수치 원장 참조.)

### 사업 리스크 3 + 대응

1. **AI 리포트 선점 창이 6~12개월 내 닫힌다.** CompanAIn 이미 GA. → LLM 브리핑을 다음 스프린트 최우선(백엔드 데이터 파이프라인 존재, 호출부만 교체 2~3주).
2. **이중언어 wedge가 코드에 없다.** → LanguageContext + `t(lang)` 일괄 치환 + 인라인 한국어 이관. 신기능 아니라 배선 1~2주. 이게 안 되면 나머지 전략 무의미.
3. **병원 무료 앱 + 순수기록 무료화로 지불의사 증발.** → 유료 후크를 "기록"이 아니라 **"내가 소유하는, 병원을 옮겨도 따라오는, 영/한 리포트"**에 둠. 이사·이민·검역 페인을 정면 마케팅. 수익 구독+커머스 이원화.

### 구독 외 대안 수익원 (4종)

1. **보험 리드 제너레이션 (호주 1순위).** vet 비용 급등·41% 의료비 패닉. 케어 앱은 보험 가입 의향이 가장 높은 순간(등록 직후, 첫 이상징후)에 유저와 만남. 호주 펫보험 리드 건당 A$20~60.
2. **커머스 제휴 (한국 1순위, 필수).** 정량 데이터(식사 g/물 ml/투약)로 사료·처방식·영양제·리필을 맥락 추천 → 제휴 커머스 수수료. "물 기록 3일째 없음 → 급수기 추천" 식 데이터 트리거.
3. **B2B2C 동물병원 채널.** 리포트 수신자(수의사)용 대시보드를 병원 SaaS로(PetDesk가 병원에 US$389/월) 또는 화이트라벨. 세일즈 사이클 길어 12개월+ 로드맵.
4. **검사키트/텔레헬스 중개.** 핏펫(검사키트)·Airvet(US$35/월) 모델.

**권고 조합:** 호주 = 구독(Plus/Family) + 보험 리드. 한국 = 커머스 로열티 + 구독 최소화. 동일 코드베이스로 수익 레이어만 지역별 스위치.

### 방어 가능한 moat (3층)

1. **데이터 moat** — 일상 다이어리에서 자동 축적된 정량 시계열(경쟁사는 vet 기록 업로드가 소스라 이벤트성). 오래 쓸수록 이관 비용↑.
2. **언어 moat** — 영/한(→중·베트남어) 이중언어 리포트. 한국어 수의학 용어→영어 임상 용어 매핑은 축적되는 도메인 자산.
3. **소유권 moat** — 병원 비종속. 이민국가 호주 + 국제이동 반려인에 정확히 맞음.

### Go / Pivot / Kill

**조건부 GO — 90일 안에 3게이트 미통과 시 KILL.** 3축(AI·이중언어·수익화)이 전부 "신규 R&D가 아니라 이미 있는 뼈대에 배선"이라 회복 가능:
- AI: 데이터 파이프라인 완성 → LLM 호출만 추가
- 이중언어: en 번역 완비 → 배선만
- 만료링크: 백엔드 완성 → 프론트 연결만
- 결제: entitlement 타입 완비 → RevenueCat 연결만

| 게이트 | 조건 |
|---|---|
| **G1 (Week 3)** | 이중언어 실동작: `t(lang)` 배선 + 인라인 한국어 이관, 영어 100% 렌더 |
| **G2 (Week 6)** | 진짜 LLM 브리핑: LLM 호출로 생성, 디스클레이머·규제 톤 검수 |
| **G3 (Week 12)** | 유료의지 검증: 베타 200명 → 트라이얼→유료 ≥12% 또는 보험/커머스 리드 클릭 ≥8% (+ vet 수용성 인터뷰 5~10명 권장, C섹션 참조) |

Kill 시 Pivot: 정량 케어 데이터 파이프라인을 B2B(병원·보험·펫푸드)에 판매 또는 종료.

**한 줄 결론:** 빈틈은 실재하지만 좁고(세그먼트+번들+타이밍), 지금 코드는 아직 못 메운다. **속도가 유일한 해자.** 6~12개월 안에 AI·이중언어 실물화 + 커머스/보험 이원화하면 린 사업 생존 확률 ~40%, 구독 단독 순수기록앱은 ~15% — 후자는 하지 말 것.

## B. 마케터 관점

### 타깃 고객 (지불의사 순)

- **P1. 아픈 반려동물 보호자** (만성질환/증상 추적자) — 핵심 유료 세그먼트, WTP 최상. vet 진료실에서 "언제부터? 하루 몇 번? 사료 얼마나?"에 답 못 하는 통증. 정량 폼이 정확히 이 데이터를 수집. **지불 트리거 = 진단 직후 2주.** 만성질환은 수년 지속 → 리텐션·LTV 최상. **예산 80% 집중.**
- **P2. 다인 가구/원격 공동 양육** (가족 케어 코디네이터) — Family 티어 타깃. **단 초대 기능이 없으면 지금 확보 불가.**
- **P3. 호주 거주 한인/아시아계** (이중언어 브릿지) — 저CAC 웨지, TAM 작음. **단 영어가 죽은 코드라 지금은 제품이 안 팔림.** 대량 획득이 아니라 저비용 초기 거점.

### 핵심 카피 (P1 주력)
- EN: **"Turn daily notes into a vet-ready brief — so your next visit is faster, clearer, and cheaper."**
- KO: **"매일의 기록이 병원용 브리핑으로 — 다음 진료를 더 짧고 정확하게."**
- vet 비용 각도: "'검사 한 번 더 해보죠' 대신, 무슨 일이 있었는지 그대로 보여주세요."
- **금지어:** "clinician-grade", "diagnosis", "진단", "의료 자문" (CompanAIn이 "clinician-grade"로 안는 규제 리스크를 피해 "vet-ready summary / 병원용 정리"로 포지셔닝).

### 채널 전략 (저예산)

**호주 (1차):**
- **ASO(제로 예산 최우선):** `pet health tracker`, `dog symptom diary`, `cat health log`, `vet visit notes`, `pet poop tracker`(롱테일 저경쟁). 제목에 "Vet-Ready Pet Health Diary". 스크린샷 1번을 "브리핑 결과물"로. **단 스토어 문구가 영어여야 하므로 언어 스위처가 전제.** 리뷰 요청은 "AI 브리핑 첫 생성" 순간(가치 실현 지점).
- **수의사 클리닉 제휴(호주 가장 결정적):** 만성질환·재진 많은 독립 vet 10곳에 QR 데스크 카드("다음 진료 전 증상 정리"). vet에게 파는 각도 = "정리된 브리핑 = 진료 시간↓ 문진 정확도↑". 만료링크 완성 후 "pre-visit brief" 워크플로우(`vet_report_viewer` 역할이 이미 이걸 위해 정의됨).
- **커뮤니티(제로 예산, P1 타깃):** 견종 서브가 아니라 **질환 서브/그룹**(r/AskVet, "Canine Kidney Disease Support", "Feline Diabetes"). 직접 광고 금지, 가치 제공형.
- **유료는 나중에:** P1 리타게팅(7일 미전환)만 소액.

**한국 (2차):** 순수 구독 대신 **P3 웨지 + 콘텐츠 SEO.** 질환별 네이버 카페 정보성 글, 재한 외국인 FB 그룹, "동물병원 가기 전 증상 정리하는 법" 유튜브 숏폼. 수익은 커머스/거래 레이어 결합(별도 BM 트랙).

### 앱 내 바이럴 루프 — 현재 하나도 실존하지 않음

- **루프 A(가족 초대, K-factor):** 초대 UI·발송·수락 코드 0건. 지금 K=0. → `Settings/Pet`에 "가족 초대" 딥링크(`pawbloom://join/{petId}/{token}`), 초대받은 사람 설치→자동 `caregiver` 조인. **초대는 무료로, 2인 동시 기록 감지 시 Family 제안**(초대에 벽을 세우면 루프가 죽음). **우선순위 1.**
- **루프 B(vet 링크 공유, 바이럴):** `MockShareCard`만, 받는 사람이 볼 웹 뷰어 없음 → 바이럴 0. → 실제 토큰+만료 웹 뷰어, 뷰어 하단 "Create your pet's brief" CTA로 획득 채널화. **우선순위 2.**
- **루프 C(주간 기록 카드):** 인스타 공유용 예쁜 주간 요약 이미지 → 오가닉 인지.

**판정:** 마케팅을 태우기 전에 루프 A(초대)를 반드시 먼저 짓는다. 초대 없는 유료 획득은 밑 빠진 독.

### 리텐션

- **투약 있는 펫만 재방문 훅(리마인더)이 있다.** 건강한 펫(대다수)은 앱이 먼저 말 거는 순간 0 → 기록 앱 자연 이탈 곡선에 무방비. 알림 문구도 한국어 하드코딩.
- 개선(우선순위): ① 저녁 기록 리마인더("오늘 하루 어땠나요? 30초면 기록") ② 증상 추적 리마인더(P1 전용) ③ 주간 브리핑 넛지("이번 주 5일 완성 — 브리핑 볼 준비") ④ 스트릭(죄책감 X) ⑤ 알림 i18n.
- **연쇄 리스크:** 체크리스트 단방향 토글·night↔memo 충돌은 초기 신뢰 파괴 → 조기 이탈. 리텐션 캠페인보다 이 버그 수정이 선행.

### 90일 마케팅 플랜 (저예산)

- **Phase 0 (W1~4) — 마케팅 태우기 전 전제 구축:** 언어 스위처+en 노출, 로컬-우선 온보딩+소셜 로그인, 체크리스트/카테고리 버그 수정, 퍼널 6단계 계측, 가족 초대 딥링크, 저녁 기록 리마인더.
- **Phase 1 (W5~8) — 오가닉 시딩·ASO:** ASO 최적화, 호주 독립 vet 10곳 제휴, 질환별 커뮤니티 시딩, 주간 카드, 1차 코호트 측정.
- **Phase 2 (W9~12) — 전환 최적화·vet 링크 루프:** 실제 공유 링크+만료 뷰어, Plus 트라이얼, 전환 트리거 정밀화(진단 후 2주/2인 감지), 최대 드롭 A/B, 90일 리뷰(퍼널 전환·K-factor·바이럴 계수·D1/D7/D30).

한국(P3)은 Phase 2 이후 별도.

## C. 교차검증 — 이 문서 고유 메타 발견

미검증 수치 원장과 규제 사각지대(TGA SaMD/AVA, vet 수용성, export 양면성, 계절성, 관측성 부재)는 [00 종합 요약](00-executive-summary.md)에 단일화했다. 여기서는 이 문서에서만 도출되는 유일한 메타 발견을 기록한다.

### 첫 타깃 세그먼트 불일치 (해소 필요)
CEO의 SOM 계산은 P3(재호 한인 SAM 3만~8만)을 분모로 쓰는데, 마케터는 P1(전체 아픈 반려동물)을 주력으로 삼는다. **누가 진짜 첫 타깃인가에 대한 합의 필요.** 권장: 유료 검증은 P1(급성 통증→즉시 전환)로, 저CAC 초기 획득은 P3 웨지로 — 역할을 분리하되 동일 세그먼트로 착각하지 말 것.
